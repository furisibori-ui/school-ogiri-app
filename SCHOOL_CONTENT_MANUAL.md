# 日本の学校公式サイト生成マニュアル
## 社会言語学的分析に基づく完全版ガイド

本マニュアルは、全国の学校サイトの文体、校訓、教育目標を体系的に分析した結果に基づき、リアルで説得力のある学校サイトを生成するための包括的なガイドです。

## 📚 第1部：校長メッセージの文体と構造

### 基本構成（伝統的な手紙形式）

| 構成ブロック | 要素 | 具体例 |
|------------|------|--------|
| **前文** | 1. 頭語 | 「拝啓」「謹啓」 |
|  | 2. 時候の挨拶・安否伺い | 「立春を過ぎ、日ごとに春の気配が感じられる今日この頃」 |
| **主文** | 3. 起こし言葉 | 「さて」「このたびは」 |
|  | 4. 本文 | 教育方針、学校行事、児童・生徒の様子 |
| **末文** | 5. 結びの言葉 | 「今後とも変わらぬご支援を賜りますよう」 |
|  | 6. 結語 | 「敬具」「謹白」 |
| **後付** | 7. 日付 | 「令和6年2月17日」 |
|  | 8. 署名 | 「校長 〇〇〇〇」 |

### 冒頭の挨拶（⚠️ 重要な注意事項）

**学校サイトは一年中同じテキストが表示されるため、特定の季節を指定してはいけません。**

#### ✅ 推奨される挨拶（一年中通用）
- 「日頃より本校の教育活動にご理解とご協力を賜り、厚く御礼申し上げます」
- 「本校ホームページをご覧いただき、誠にありがとうございます」
- 「皆様にはますますご清祥のこととお慶び申し上げます」
- 「保護者の皆様、地域の皆様におかれましては、日頃より本校の教育活動に温かいご支援を賜り、心より感謝申し上げます」

#### ❌ 使用してはいけない表現（季節を特定する）
- 「立春を過ぎ」「桜の花は今を盛りと」「三寒四温の候」
- 「青葉若葉の眩しい」「紅葉の季節を迎え」
- その他、具体的な季節や月を示す表現全般

#### 📝 参考：校長だよりの時候の挨拶（季節別）

以下は学校だより等で使用される季節別の挨拶例ですが、**公式サイトでは使用しません**：

- 2月：「立春を過ぎ、日ごとに春の気配が感じられる今日この頃」
- 4月：「桜の花は今を盛りと咲き誇っております」
- 7-8月：「青葉若葉の眩しい季節となりました」
- 11月：「小春日和の好季節、紅葉の美しい季節を迎えました」

### 文体の特徴

1. **丁寧語の多用**
   - 「〜でございます」（断定を避ける品格）
   - 「〜してまいりました」（継続的な努力の強調）
   - 「〜させていただいております」（謙虚な姿勢）

2. **具体的な学年別活動への言及**
   - 1年生：「ひらがなの学習」「毎日の登校」
   - 2年生：「ミニトマトの栽培」「1年生への学校探検案内」
   - 3年生：「リコーダー」「毛筆」「理科実験」
   - 4年生：「クラブ活動開始」「運動会での演技」
   - 5年生：「林間学校」「調理実習」「学校行事の裏方」
   - 6年生：「修学旅行」「1年生の世話」「学校の牽引」

3. **地域との連帯感**
   - 「地域の皆様のご協力により」
   - 「〇〇商店街の方々と」
   - 「創立150周年という長い歴史の中で」

4. **現代的価値観の統合**
   - 「自己有用感」「自己肯定感」
   - 「多様性を認め合い」
   - 「SDGs（持続可能な開発目標）」
   - 「食品ロスの削減」

### 忌み言葉の厳格な回避

慶事の手紙では以下の言葉を**絶対に使用しない**：
- 切れる、終わる、死、苦、失う、枯れる、倒れる、衰える、滅びる

## 🎵 第3部：校歌の作詞マニュアル

### 基本構造（3番構成）

#### リズム：七五調または八六調
- 七五調：7文字・5文字（「朝日輝く／この地に」）
- 八六調：8文字・6文字（「〇〇駅から／通う道」）

### 各番の構成と必須要素

#### 1番：地域の景観・自然描写

**必須要素**：
1. 具体的なランドマーク名（2つ以上）
2. 自然環境（山、川、海、空、緑、風、光）
3. 時間帯の描写（朝日、夕日、星空）
4. 季節感（桜、新緑、紅葉、雪）

**例文**：
```
朝日輝く 〇〇に
[ランドマーク名] 仰ぎて 学び舎あり
〇〇の風 薫る中
若き我らの 歌声響く
```

#### 2番：歴史と伝統、校訓の織り込み

**必須要素**：
1. 地域の歴史的背景（創立年代、地域の発展）
2. 創立の理念（創立者の志）
3. 校訓の四字熟語を自然に織り込む
4. 学校の誇り

**例文**：
```
[明治/大正/昭和]の 昔より
この地に根ざし 学びの灯
[校訓の言葉] 胸に秘め
真理の道を 進みゆく
```

#### 3番：未来への誓い、母校への愛

**必須要素**：
1. 生徒たちの決意（「拓く」「進む」「輝く」などの動詞）
2. 地域への貢献（「この地を」「世界へ」など）
3. 母校への永遠の愛（「ああ 〇〇学校」など）
4. 結びの力強い言葉

**例文**：
```
[地域の言葉] の 空の下
友と励まし 学ぶ日々
未来を拓く 若き力
ああ [学校名] 栄えあれ
```

### 使用すべき伝統的表現

- 語尾：「〜あり」「〜ゆく」「〜あれ」「〜かな」
- 主語：「我ら」「若き」「友と」
- 感嘆：「ああ」「今こそ」
- 動詞：「仰ぐ」「拓く」「進む」「輝く」「響く」「学ぶ」

## 👨‍🏫 第4部：教員紹介の個性化マニュアル

### 文字数と構成

- **文字数**：250-350字（短い紹介文は厳禁）
- **構成**：勤務年数 → 地域連携エピソード → 教育哲学 → 生徒とのエピソード

### 科目別の書き分け例

#### 国語科
```
本校に15年勤務しております国語科教員でございます。毎週金曜日には、
本校から徒歩5分の[図書館名]にて古典文学読書会を開催し、地域の方々
にも広く参加いただいております。また、[商店街名]の[書店名]と連携し、
生徒たちの読書感想文を店頭に展示するプロジェクトを10年以上継続して
おります。「言葉は心の窓である」をモットーに、日本語の美しさと表現
の豊かさを生徒たちに伝えることに情熱を注いでおります。昨年度は、
[地域の祭り名]で生徒たちが方言研究の成果を発表し、大変好評をいただ
きました。
```

#### 理科
```
本校に8年勤務しております理科教員でございます。[川名]での水質調査、
[公園名]での植物観察、[山名]での地層研究など、周辺の豊かな自然環境
を最大限に活用した実践的な授業を展開しております。特に、[施設名]と
連携した環境保全プロジェクトでは、生徒たちが自ら課題を発見し、解決
策を提案する主体的な学びを実現しております。「理科は暗記ではなく発
見である」という信念のもと、生徒一人ひとりの探究心を育むことを大切
にしております。昨年度は、[地域イベント名]で生徒たちの研究成果を発
表し、地域住民の方々から高い評価をいただきました。
```

## 🏛️ 第5部：歴史的建物の説明マニュアル

### 基本構成（100-150字）

1. **建築年代**：「明治○○年〜大正○○年」（具体的な年号）
2. **建築様式**：「木造平屋建て」「瓦葺き」「白壁」など
3. **地域との位置関係**：「坂の上に」「川沿いに」「旧街道沿いに」
4. **時代背景**：当時の地域の産業、戦前の様子など
5. **地域住民との関わり**：寄付、建設協力など
6. **歴史的エピソード**：開校式の様子、著名人の訪問など

### 例文

#### 初代校舎
```
明治35年、[ランドマーク名]の麓に建てられた木造平屋建ての校舎でござ
います。当時、この地域は[産業名]が盛んで、地域住民の皆様からの寄付
により建設されました。創立者[名前]先生の「[地域名]の精神を受け継ぎ、
未来を切り拓く人材を育てる」という教育理念のもと、多くの子どもたち
が学びました。大正12年の[災害名]により惜しくも焼失いたしましたが、
その精神は今も本校に受け継がれております。
```

## 🏅 第8部：校章（エンブレム）のデザイン原則

### 校章の役割と重要性

校章は学校の精神や目標を視覚的なシンボルとして凝縮したものです。制服、学生証、校舎の銘板などに掲げられ、所属する児童・生徒のアイデンティティを形成する重要な役割を担います。

日本の学校における校章デザインは、古来の家紋文化や神話的象徴をルーツに持ちながら、それぞれの時代背景や教育的願いを反映して独自の進化を遂げてきました。

### 伝統的象徴：三種の神器の教育的解釈

日本神話に由来する「三種の神器（鏡、剣、勾玉）」は、教育的な徳目と結びついて解釈されます。

| 神器 | 象徴する徳 | 意味 | 実例 |
|------|----------|------|------|
| 鏡（八咫鏡） | 知恵（知） | 自己を映し出し、常に内面を省みる姿勢 | 名古屋女学校（大正8年制定） |
| 剣 | 勇気（勇） | 決断力、正義を貫く強さ | - |
| 勾玉 | 思いやり（仁） | 慈しみ、他者への配慮 | - |

**鏡のモチーフ**が特に多く用いられるのは、「鏡のごとく清らかな人づくり」という願いが込められているためです。

### 自然物・動植物モチーフに託す成長の願い

家紋の伝統を受け継ぎ、自然の草木や動物の意匠を取り入れたデザインも多数存在します。

| モチーフ | 象徴する意味 | 実例 |
|---------|------------|------|
| 鶴 | 高潔さ、品格、飛躍 | 神奈川県立上鶴間高等学校「大空に舞い上がる若鶴」 |
| 葦（あし） | 強靭さ、質実剛健 | 関西大学「淀川の葦」 |
| 桜 | 日本の美、春の訪れ、新しい始まり | - |
| 松 | 不変、長寿、永続性 | - |
| 梅 | 高潔、忍耐、早春の希望 | - |
| 竹 | 柔軟性、成長力、清廉 | - |

### 幾何学模様と抽象概念の具象化

シンプルな幾何学模様は、人間の深層意識に特定のメッセージを呼び起こすアーキタイプ（原型）として機能します。

| 図形 | 象徴する意味 | 実例 |
|------|------------|------|
| 円 | 永遠、調和、心のバランス、完全性 | - |
| 三角形 | 安定した基盤、天への上昇、創造の意欲、均衡と力 | 千葉県立小金高等学校「三角形を積み重ねた図案」 |
| 盾型 | たくましい精神力、根気強さ、協力と和 | 昭和22年創立校 |
| 多角形 | 多様性、複雑性の調和 | - |

### 時代精神の反映

| 時代 | デザインの特徴 | 実例 |
|------|-------------|------|
| 明治時代 | 西洋文化と日本伝統の融合、進取の気性 | 新潟県立新潟商業高等学校（明治38年）「ヘルメスの杖+N・Cイニシャル」 |
| 大正・昭和初期 | シンプルで力強い、国家主義的要素 | - |
| 戦後 | 民主的、平和的、開放的 | - |
| 現代 | 国際性、多様性、具体的な教育メソッドの視覚化 | New International School「2つの翼＝デュアル・ランゲージ」 |

### 色彩の象徴性

| 色 | 意味 | 用途 |
|----|------|------|
| 紺（Navy Blue） | 伝統、誠実、知性 | 最も一般的な背景色 |
| 金（Gold） | 栄光、高貴、卓越 | 文字、刺繍 |
| 白（White） | 清浄、純粋、平和 | 文字、背景 |

### 地域との結びつき

校章は地域の特性を反映すべきです：

| 地域特性 | 推奨モチーフ | 色彩 |
|---------|------------|------|
| 山間部 | 山岳、樹木、鷹 | 緑、茶色 |
| 海沿い | 波、船、灯台 | 青、白 |
| 歴史地区 | 城、寺社、伝統文様 | 茶色、金 |
| 商業地 | 商船、ヘルメスの杖、歯車 | 赤、金 |
| 工業地帯 | 歯車、工具、炎 | グレー、オレンジ |

### 校章生成プロンプトの構成要素

優れた校章プロンプトには以下の要素を含めます：

1. **基本形状**：円形、盾型、三角形など
2. **中心モチーフ**：三種の神器または地域の象徴
3. **周辺装飾**：地域のランドマーク、動植物
4. **色彩指定**：伝統色+地域色
5. **文字要素**：校名の頭文字、創立年
6. **スタイル指定**：伝統的、家紋風、対称的など

**例**：
```
A traditional Japanese school emblem in circular shield form, 
featuring a stylized mirror (yamatanokagami) at the center 
symbolizing wisdom, surrounded by [富士山] mountain peaks motif 
in gold embroidery on a navy blue background with white crane wings, 
kanji character [富] in gold at the top, established year [1905] 
at the bottom, family crest style, old-fashioned design, 
detailed traditional Japanese heraldry, symmetrical composition
```

## 📷 第10部：行事写真の「生っぽさ」を再現する撮影原則

### 学校ブログに掲載される写真の特徴

学校公式ホームページやブログに掲載される行事写真は、商業広告のような完璧な写真ではなく、「飾らないリアルで生っぽい雰囲気」を持ちます。以下の原則に基づいて撮影・生成します。

### 1. 画角と構図の傾向

#### アスペクト比
- **横長（16:9 または 4:3）を推奨**
- 理由：ホームページでの配置やスクロール時の見やすさ
- 縦型写真は避ける

#### 画角の「引き」
- **被写体を極端なアップにしない**
- **周囲の状況がわかる適度な引きの画角**
- 理由：後からウェブサイトのデザインに合わせてトリミングや加工ができる
- 背景（教室、体育館、グラウンドなど）が適度に写り込む

### 2. 被写体（生徒・教員）の配置

#### 密着感の演出
- **被写体同士の物理的距離を近づける**
- 写真上の余白を減らして一体感や活気を出す
- 「和気藹々とした雰囲気」を視覚化

#### 撮影アングル
- **斜め方向から撮影**して距離を圧縮
- 複数の生徒が自然に集まっている様子
- 小グループでの会話や協力の瞬間

### 3. 自然な目線（最重要）

#### ❌ 避けるべき表現
- 全員がカメラを見ている「作られたポーズ」
- わざとらしい笑顔や決めポーズ
- スタジオ撮影のような完璧な構図

#### ✅ 推奨される表現
- **行事そのものに集中している様子**
- 会話している自然な視線（右向き、左向きなど）
- 作業に取り組む真剣な表情
- 友人との自然な笑顔や会話
- 「撮られていることを意識していない瞬間」

### 4. 背景と環境の整理

#### 文脈との一致
- 「今日は〇〇をしました」という活動報告の文章に沿った写真
- 具体的な行動の瞬間を切り取る
- 環境（場所、道具、季節感）が明確にわかる

#### 環境の写り込み
- 教室：黒板、机、掲示物
- 体育館：バスケットゴール、ステージ
- グラウンド：校舎、木々、空
- 地域施設：具体的なランドマークや建物

### 5. 行事別の推奨構図

| 行事 | 推奨構図 | 視線 | 背景 |
|------|---------|------|------|
| 体育祭準備 | 生徒が集まって話し合い | 互いを見合う | 体育館、用具 |
| 文化祭準備 | 制作作業中の様子 | 作業に集中 | 教室、材料 |
| 理科実験 | 実験器具を囲む | 実験に注目 | 理科室、機材 |
| 調理実習 | 調理台を囲む | 料理に集中 | 調理室、食材 |
| 修学旅行 | ランドマーク前で談笑 | 互いに会話 | 観光地 |
| 卒業式 | 整列しているが一部談笑 | 前方と横 | 体育館、ステージ |
